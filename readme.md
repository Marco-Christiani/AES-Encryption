<a name="install"></a>
## Installation
```
unzip AES-Encryption.zip
cd AES-Encryption
pipenv install
```
If pipenv is not installed, install with ```pip install pipenv```

<a name="cmdline"></a>
## Command Line Tool
To read from text and key from files:
```
python cipher.py -tf <path/to/textfile> -kf <path/to/keyfile>
```

To pass literal strings:
```
python cipher.py -tf <path/to/textfile> -kf <path/to/keyfile>
```
